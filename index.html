Kaboom Demo Games

Welcome!

This is a site to showcase three games I have made using the JavaScript library Kaboom.js.

Snek

This is a classic snake game, like what was present on the old Nokia phones. It uses a linked list to grow the tail and ensure all segments follow the path of their parent.

Pew Pew

A space-shooter in the style of Space Invaders.
